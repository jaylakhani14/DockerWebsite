Basic Docker Linux Commands

Basic Run Command:

docker run [name of docker]

Run a specific container:

docker container run --name [container_name] [name of docker]

To check if it is running: docker ps -a

Run a command in background mode:

docker container run -d [name of docker]

Start a container

dockerrun-itubuntu:latest/bin/bash

Proper way to run Docker container commands

docker-compose run [name_of_container] [command]
